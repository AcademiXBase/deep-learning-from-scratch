# Chapter 5: 誤差逆伝播法

## 問いと計算グラフ

p.124 \
問1：太郎くんはスーパーで1個100円のリンゴを2個買いました。支払う金額を求めなさい。ただし、消費税が10%適用されるものとします。

順伝播：

```mermaid
flowchart LR
    A["apple<br/>(100)"] -->|100| C(("×"))
    B["apple_num<br/>(2)"] -->|2| C
    C -->|200| D(("×"))
    E["tax<br/>(1.1)"] -->|1.1| D
    D -->|220| F["price<br/>(220)"]
```

逆伝播：

```mermaid
flowchart RL
    C(("×")) --> |2.2| A["apple<br/>(100)"]
    C --> |110| B["apple_num<br/>(2)"]
    D(("×")) --> |1.1| C
    D --> |200| E["tax<br/>(1.1)"]
    F["price<br/>(220)"] --> |1| D
```

p.125 \
問2：太郎君はスーパーでりんごを2個、みかんを3個買いました。りんごは1個100円、みかんは1個150円です。消費税が10%かかるものとして支払う金額を求めなさい。

順伝播：

```mermaid
flowchart LR
    A["apple_num<br/>(2)"] -->|2| C
    B["apple<br/>(100)"] -->|100| C(("×"))
    C -->|200| G(("＋"))

    D["orange<br/>(150)"] -->|150| E(("×"))
    F["orange_num<br/>(3)"] -->|3| E
    E -->|450| G

    G -->|650| H(("×"))
    I["tax<br/>(1.1)"] -->|1.1| H
    H -->|715| J["price<br/>(715)"]
```

逆伝播：

```mermaid
flowchart RL
    C(("×")) -->|110| A["apple_num<br/>(2)"]
    C -->|2.2| B["apple<br/>(100)"]
    G(("＋")) -->|1.1| C

    E(("×")) -->|3.3| D["orange<br/>(150)"]
    E -->|165| F["orange_num<br/>(3)"]
    G -->|1.1| E

    H(("×")) -->|1.1| G
    H -->|650| I["tax<br/>(1.1)"]
    J["price<br/>(715)"] -->|1| H
```
